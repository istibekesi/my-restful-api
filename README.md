my-restful-api
==============

## My RESTful API demo

Demo appliction demonstrating RESTful API Design.

----------

### Requirements

* Java 1.6+
* Maven 3.0+

### How to install

1. Build
`mvn install`
2. Run
`mvn spring-boot:run`
3. Try
`http://localhost:8080/myrest/hello`
`http://localhost:8080/myrest/v1/manager/1`
`http://localhost:8080/myrest/v2/manager/1`
`...`
